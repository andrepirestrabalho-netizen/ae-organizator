# AE Organizator

ScriptUI Panel for Adobe After Effects that automatically organizes project assets into folders.

## Features

- Organizes COMPS
- Organizes PRE-COMPS
- Organizes AUDIO
- Organizes IMAGES
- Organizes VIDEOS
- Organizes SOLIDS

## Installation

Copy the file to:

C:\Program Files\Adobe\Adobe After Effects\Support Files\Scripts\ScriptUI Panels

or run the installer.

## Compatibility

Tested with Adobe After Effects 2024 / 2025 / 2026
